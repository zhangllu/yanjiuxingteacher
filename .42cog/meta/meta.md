## 项目描述

活水智聊教学演示版（42chatdemo）是一款面向个人用户与小型团队设计的基于 Web 的多模型 AI 对话中枢，为频繁在不同模型间工作的个人用户提供统一入口。它将会话管理、模型切换、提示词模板、联网搜索与数据同步收拢到一致的交互中。在强制登录与基础角色的保障下，用户无需改变既有习惯即可获得更连贯的对话体验：对话被持续保存、可被检索，也可被导出；配置集中收纳、随时可测、按需启用。

## 目标用户

- 多模型重度用户：频繁在不同 AI 模型间切换工作的个人用户
- 小型团队：需要统一管理 AI 对话与配置的小型团队

## 核心价值主张

1. **统一入口**：聚合登录、配置与切换，在单一界面完成跨模型交流
2. **对话管理**：对话持续保存、可被检索、可被导出，保留知识积累
3. **智能增强**：提示词模板与联网搜索，按需获取实时信息
4. **数据安全**：云端同步与本地兜底，降低切换成本

## 技术栈

**前端**:
- 框架: Next.js 15
- UI: Tailwind CSS + shadcn/ui
- 语言: TypeScript

**后端**:
- 运行时: Bun / Node.js
- 数据库: PostgreSQL
- ORM: Drizzle ORM

**基础设施**:
- 托管: Vercel / EdgeOne Pages
- 数据库: Neon / Supabase
- 认证: Better Auth

**AI集成**:
- AI SDK: Vercel AI SDK
- 模型: Claude / GPT-4 / 多模型支持
