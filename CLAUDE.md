# Claude 使用指南

## 个人环境配置

### 硬件环境
- **设备**：搭载 Apple 芯片的 Mac 电脑

### 开发环境
- **Node.js 管理**：通过 bun 进行安装管理
- **Python 环境**：通过 uv 配置
- **Git 代码托管平台**：cnb.cool

## 沟通规范

所有与 Claude 的沟通和回复都使用中文进行。

## 工作流程

### 项目结构
本项目采用模块化结构，主要目录包括：
- `.42cog/` - 认知框架和工作流程
- `.42plugin/` - 插件和教育资源
- `spec/` - 项目规格说明
- `src/` - 源代码
- `notes/` - 笔记和文档
- `scripts/` - 脚本工具

### 开发规范
1. 使用 bun 管理 Node.js 依赖和运行脚本
2. 使用 uv 管理 Python 环境和依赖
3. 代码托管在 cnb.cool 平台
4. 遵循项目既定的目录结构和命名规范

## 注意事项

- 所有文档和代码注释优先使用中文
- 保持代码简洁，避免过度设计
- 遵循 Apple Silicon 架构的最佳实践
- 充分利用 bun 和 uv 的性能优势
